# production/cyber-otter-2077/

The polished edition keeps the Codex-compatible 11-row geometry and state semantics
while using a compact, non-pixel mascot rendering. Its claim is structural, not
official: compatibility is established by the atlas contract and QA ledger, not by
visual resemblance or endorsement.
精制版保留 Codex 兼容 11 行几何与状态语义，采用紧凑的非像素吉祥物渲染。其主张是结构性
的，不是官方性的：兼容由图集契约与 QA 账本建立，而非由视觉相似或背书建立。

The reference image, prompts, approved strips, state frames, final atlas, frozen
release cells, and QA evidence remain in this directory. Rejected attempts do not.
参考图、提示词、获准条带、状态帧、终稿、冻结发布单元与 QA 证据均留在本目录；失败稿
不在其中。
