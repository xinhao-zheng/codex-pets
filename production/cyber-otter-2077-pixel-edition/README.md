# production/cyber-otter-2077-pixel-edition/

The pixel edition is authored at the native `192 × 208` cell scale. Stepped
contours, bounded value ramps, and discrete clusters carry the style; enlargement
of a coarse mock grid does not.
像素版直接在原生 `192 × 208` 单元尺度上成立。阶梯轮廓、受限明度阶与离散色块承载其
风格；把粗网格放大，并不能。

The reference image, prompts, approved strips, state frames, final atlas, frozen
release cells, and QA evidence remain in this directory. Rejected attempts do not.
参考图、提示词、获准条带、状态帧、终稿、冻结发布单元与 QA 证据均留在本目录；失败稿
不在其中。
